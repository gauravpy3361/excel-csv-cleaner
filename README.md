# 📊 Excel/CSV Automation Toolkit (Python)

💼 Built to automate real-world Excel/CSV tasks and reduce hours of manual work into seconds.

A Python-based automation tool for handling real-world Excel and CSV data tasks like cleaning, formatting, merging, and removing duplicates.

---

## 🚀 What this tool does

This project helps automate repetitive data work such as:

* Cleaning messy datasets
* Removing duplicate rows
* Formatting Excel/CSV files
* Merging multiple CSV files into one

---

## 💡 Why this is useful

Manual Excel work is:

* Slow
* Repetitive
* Error-prone

This tool:

* Saves time
* Reduces human error
* Handles large datasets efficiently

---

## 🛠️ Features

### 🔹 Auto Format

* Cleans column names
* Removes empty rows
* Strips extra spaces
* Formats date columns

### 🔹 Remove Duplicates

* Detects and removes duplicate rows
* Outputs clean dataset

### 🔹 Merge CSV Files

* Combines multiple CSV files into one
* Tracks source file for each row

### 🔹 Full Data Cleaning

* Combines multiple cleaning steps into one process

---

## ⚙️ Installation

1. Install Python (if not installed)
2. Install required library:
   pip install pandas

---

## ▶️ How to Use (CLI)

### Format file

python main.py --task format --input test.csv

### Remove duplicates

python main.py --task dedupe --input test.csv

### Merge files

python main.py --task merge --folder my_folder

### Full clean

python main.py --task clean --input test.csv
